# LinkedIn — Headline + About + Open to Work

**Cómo usar este archivo:**
1. Copiar el bloque "Headline" → pegar en LinkedIn (perfil → editar headline)
2. Copiar el bloque "About" → pegar en LinkedIn (perfil → editar sección About)
3. Configurar "Open to Work" siguiendo la guía al final

---

## 1) Headline

Pegar exactamente esto (199 caracteres, máx LinkedIn = 220):

```
AI Automation Specialist | n8n · Vapi · Claude · OpenAI | Co-founder @ Nexo-Terra | Building intelligent workflows for SMEs | Available for remote roles worldwide | EN/ES
```

**Por qué este headline funciona:**
- Empieza con el rol exacto que buscan recruiters
- Lista keywords técnicos (recruiters filtran por "n8n", "Vapi", "Claude")
- Menciona Nexo-Terra (credibilidad: tenés agencia)
- "Available for remote roles" = activador para sourcers
- "EN/ES" = filtro positivo para empresas bilingües
