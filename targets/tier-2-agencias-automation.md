# Tier 2 — Agencias de AI Automation (USA/UK/AU con team LatAm o async-first)

**Estrategia:** estas agencias contratan operadores técnicos para implementar automatizaciones para sus clientes. Tu trabajo es ejecutar (no vender). Inglés intermedio alcanza si la cultura es async-first.

**Cómo investigar cada una (5 min c/u):**
1. ¿Tienen página de carreras / "We're hiring"?
2. ¿Aparecen ofertas compatibles (AI engineer / automation / integration / n8n / Vapi)?
3. ¿Mencionan que contratan LatAm o remote international?
4. ¿El stack mencionado matchea con el tuyo?
5. Si todo OK → cargar en Airtable + aplicar.
